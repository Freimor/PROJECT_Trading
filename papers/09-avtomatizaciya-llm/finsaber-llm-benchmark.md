# FINSABER Benchmark

Первый системный фреймворк для bias-free оценки LLM investing strategies.

## Ключевые выводы

- 20 лет данных, широкая вселенная символов
- Устранение survivorship, look-ahead, data-snooping bias
- LLM alpha **исчезает** при robust long-term тестировании
- Плохой risk control: консервативны в bull, агрессивны в bear
