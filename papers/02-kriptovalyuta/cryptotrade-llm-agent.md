# A Reflective LLM-based Agent to Guide Zero-shot Cryptocurrency Trading

**CryptoTrade** — агент на базе GPT-4/GPT-4o с рефлексией для ежедневных торговых решений по BTC, ETH, SOL.

## Ключевые результаты

- Zero-shot режим без fine-tuning на валидации
- Сравнение с Informer, PatchTST и MACD
- На бычьем периоде ETH: +22.59% Buy&Hold vs ~25.6% CryptoTrade
