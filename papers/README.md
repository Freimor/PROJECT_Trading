# Papers — библиотека источников

Коллекция **проверяемых** научных работ и профильных публикаций по финансам, трейдингу, ценным бумагам и криптовалютам.

## Критерии отбора

| Критерий | Значение |
|----------|----------|
| Период | Публикации **2021–2026** (не старше 5 лет на июль 2026) |
| Верификация | Каждая запись содержит прямую ссылку (DOI, SSRN, arXiv, официальный сайт) |
| Типы | Peer-reviewed журналы, working papers (NBER, SSRN), arXiv preprints, аналитика бирж и СМИ |
